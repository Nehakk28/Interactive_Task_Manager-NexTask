# NexTask – Interactive Task Manager

A modern and responsive task management web application built using HTML, CSS, and JavaScript. NexTask helps users organize tasks efficiently with features like task creation, editing, deletion, filtering, priority management, due dates, dark mode, task analytics, and local storage persistence.

## Features

1. Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as complete/incomplete
- Search tasks instantly
2. Task Organization
- Priority Levels (High, Medium, Low)
- Due Date Management
- Filter Tasks: (All Tasks, Active Tasks, Completed Tasks)
- Priority-based Filtering
3. Dashboard Analytics
- Total Tasks Count
- Completed Tasks Count
- Pending Tasks Count
- Completion Percentage
- Task Progress Bar
4. Calendar & Deadlines
- View upcoming deadlines
- Track tasks with due dates
5. Statistics
- Task Distribution Chart
- Completion Progress Analytics
6. Data Persistence
- Save tasks using localStorage
- Restore tasks on page reload
- Export tasks as JSON backup
- Import tasks from backup file
7. User Interface
- Responsive Design
- Modern Dashboard Layout
- Dark/Light Mode Toggle
- Smooth Animations and Transitions

## 📖 How to Use

- Add Task
- Enter task name.
- Select priority level.
- Choose due date.
- Click Add Task.
- Complete Task: Click the checkbox beside a task.
- Edit Task: Click the edit button (✏️).
- Delete Task: Click the delete button (🗑️).
- Filter Tasks : Use All, Active, Completed
- Click Export Tasks to download a JSON file.
- Restore Tasks: Click Import Tasks and select a backup file.
